Text Styles.

H1: Main page heading.

  • Font-weight: 700
  • Font-size: 30px
  • Font-family: Glegoo

H2: A page subheading.

  • Font-weight: 700
  • Font-size: 25px
  • Font-family: Open_Sans

P : paragraph text.

  • Font-weight: 400
  • Font-size: 22px
  • Font-family: Playfair

Div : a container that divides an HTML into sections and is short for "division".

  • Font-weight: bold
  • Font-size: 25px
  • Font-family: Poppins
  • Letter-spacing: 0.1rem

Ul : unordered list.

Li : list items.

  • Font-weight: 400
  • Font-size: 22px
  • Font-family: PT_Serif

Img src : The image element.

  • Font-weight: 300
  • Font-size: 22px
  • Font-family: Raleway

Br : Line Break.

  • Font-weight: 400
  • Font-size: 25px
  • Font-family: Ubuntu Mono

Span : Container to separate text for styling purposes.

  • Font-weight: 400
  • Font-size: 22px
  • Font-family: Lato

A : Anchor to create hyperlinks.
Href: the hyperlink reference that goes after 'a'.

  • Font-weight: 500
  • Font-size: 30px
  • Font-family: Great Vibes
  • Letter-spacing: 0.2rem
